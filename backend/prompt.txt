CRITICAL SCOPE — WAVELENGTH-DOMAIN FOM ONLY:
1. This dataset only accepts FOM defined as (refractive-index sensitivity) / (linewidth), i.e. FOM = S/FWHM, with units of RIU^-1 and sensitivity S in nm/RIU (or µm/RIU, converted to nm/RIU).
2. Some papers define an ADDITIONAL, DIFFERENT "FOM" that is NOT wavelength/sensing-based (e.g. FOM = Q-factor x Amplitude, which is dimensionless and unrelated to refractive-index sensitivity). This is a DIFFERENT quantity, even though the authors may also call it "FOM". Do NOT report it as the "Wavelength-based FOM" value — treat it as out of scope (see "FOM Domain" below).
3. Always record the author's own definition of FOM in "FOM Definition", verbatim or lightly paraphrased (e.g. "FOM = S/FWHM"), or null if not explicitly given.

MULTIPLE MODES/CASES — ONE RECORD PER MODE:

3. Sort alphabetically, joined with ";" (e.g., "Dielectric;Metal;Semiconductor").
4. A material's class is determined by its own physical/chemical nature (e.g., Si3N4, SiO2, and Ta2O5 are dielectrics, not semiconductors — do not classify a compound as "Semiconductor" solely because it contains a semiconductor element). If genuinely unsure, prefer the classification the paper itself uses to describe the material (e.g., "phase-change material", "all-dielectric") over your own inference, and note the uncertainty in "Notes".
